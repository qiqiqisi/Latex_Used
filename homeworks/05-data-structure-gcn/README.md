# 05 · 数据结构与 GCN 🕸️

这份综合实验把传统图数据结构与图卷积神经网络连接起来，内容包括邻接矩阵、邻接表、DFS/BFS、稀疏矩阵乘法和两层 GCN 前向传播。

## ✨ 看点

- 使用 TikZ 直接绘制算法流程图和 6 节点拓扑图
- 使用 `listings` 展示 C++ 稀疏矩阵与 GCN 核心代码
- 包含复杂度分析、调试记录、矩阵结果和实验总结
- 页眉、封面、目录、公式和代码样式比较完整

## 📦 文件说明

- `main.tex`：原始完整报告源码，也是模板分支的设计基础
- `ouc.png`、`ouc_font.png`：封面和页眉图片
- `assert/`：原仓库用于 README 展示的报告效果截图
