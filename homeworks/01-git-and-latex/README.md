# 01 · Git 与 LaTeX 入门 🐙

这份实验报告记录了 Git 常用命令、GitHub 基本操作，以及标题、章节、目录、公式和代码环境等 LaTeX 基础用法。

## 📦 文件说明

- `main.tex`：完整 LaTeX 源码
- `sample.bib`：原始参考文献示例文件
- `*.png`：命令运行过程和 LaTeX 效果截图
- `report.pdf`：已有的完整成品报告

## 🧹 整理说明

旧的独立分支中，`main.tex` 留有未解决的 Git 合并冲突。本文件夹采用仓库里内容更完整、没有冲突、同时包含 Git 与 LaTeX 两部分的版本，避免把损坏源码继续作为主文件。
